# FlockingA simple implementation of flocking for a large network of aerial vehicles. Based on the (elegant) formulation proposed in:Reza Olfati-Saber,"Flocking for Multi-Agent Dynamic Systems: Algorithms and Theory", IEEE Transactions on Automatic Control, Vol. 51 (3), 3 Mar 2006.## Initial results<p float="center">  <img src="https://github.com/tjards/flocking_network/blob/master/Figs/animation_worx.gif" width="45%" /></p><div align="left">
